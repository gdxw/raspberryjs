# raspberryjs （学习框架搭建）

## raspberryjs 介绍

raspberryjs 是给予egg开发的企业级框架，主要满足web开发中，前后端分离的开发需求。

* UI组件库
* 日志管理系统
* 项目监控系统
* 中心配置系统
* soa后端服务接入
